# Pakistani Sign Language Translator

The **Pakistani Sign Language Translator** is an AI-powered project designed to bridge the communication gap between the deaf/mute community and the hearing population. It enables real-time recognition and translation of Pakistani Sign Language (PSL) into text, making communication more inclusive and accessible.
